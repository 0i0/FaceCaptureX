# FaceCaptureX

取得iPhone X臉部追蹤資訊，並匯到Houdini. 需要iPhone X跟Houdini 16.5，會需要16.5是因為使用了新的folder parm。

Get iphone X face motion data then import to Houdini.  iPhone X, Houdini **16.5** required(due to new "folder" parmtype).
